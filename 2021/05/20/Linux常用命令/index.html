Linux常用命令

导语

因为在日常工作中会遇到远程控制、部署程序的问题，总会使用到 linux 的相关命令，所以在此总结一些常用的 linux 命令。

命令的组成

常用命令

ls

【作用】：列出目录内容

tree

【作用】：显示树形目录

需要 yum install -y tree 安装

cd

【作用】：更改当前目录

mkdir

【作用】：在当前目录下创建下一级目录

rmdir

【作用】：仅可以删除空白目录（不可以删除包含内容的目录）

touch

【作用】：创建空白文件

rm

【作用】：删除文件

  • 删除文件
  • 删除目录

删除目录和文件时，先删除文件在删除目录

cp

【作用】：源文件始终不变，仅仅是对目标文件进行改变

mv

【作用】：mv命令使源文件的状态发生改变

若果目录存在，则会将原目录移动到目标目录下；如果目录不存在，则相当于移动并重命名

cat

【作用】：正序查看文件内容

  • 参数-A 查看所有内容，$代表每行的结尾
  • 参数-n 显示行号

全文搜索

从前往后搜索：可以输入 /（其中代表要搜索的内容），搜索到的内容会高亮度显示

tail

【作用】：用于查看文件的内容，有一个常用的参数 -f 常用于查阅正在改变的日志文件

chmod

【作用】：更改当前目录权限

tar

【作用】：打包与解压

ifconfig

【作用】：查看网络地址

ps

【作用】：查看进程

kill

【作用】：结束进程

ping

【作用】：查看与此IP地址的连接情况

clear

【作用】：快速清屏

firewalld
查看防火墙状态
开启防火墙
关闭防火墙
设置对外开放访问的端口
重新载入
删除
查看已开放的端口号
vim 编译
